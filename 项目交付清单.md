## 🔌 API接口清单

### 1. 获取用户进度
```
GET /api/exam/progress/{userId}
```
**功能：** 获取用户的学习进度
**返回：** ExamProgress对象

### 2. 提交挑战答案
```
POST /api/exam/challenge/submit
```
**功能：** 提交挑战答案并验证
**参数：** ChallengeSubmission
**返回：** ChallengeValidationResult

### 3. 完成知识点
```
POST /api/exam/topic/{topicId}/complete
```
**功能：** 标记知识点为已完成
**返回：** 更新后的ExamProgress

### 4. 完成关卡
```
POST /api/exam/level/{levelId}/complete
```
**功能：** 完成关卡并获得奖励
**返回：** 更新后的ExamProgress

### 5. 获取统计信息
```
GET /api/exam/stats/{userId}
```
**功能：** 获取学习统计数据
**返回：** ExamStats对象

### 6. 重置进度
```
POST /api/exam/progress/reset
```
**功能：** 重置用户学习进度
**返回：** 成功消息
