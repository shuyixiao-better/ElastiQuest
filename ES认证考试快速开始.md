# ES认证工程师考试学习系统 - 快速开始

## 🚀 5分钟快速体验

### 1. 启动项目

#### 方式一：使用启动脚本（推荐）

**Windows:**
```bash
start.bat
```

**Mac/Linux:**
```bash
chmod +x start.sh
./start.sh
```

#### 方式二：手动启动

**启动后端：**
```bash
cd backend
mvn spring-boot:run
```

**启动前端：**
```bash
cd frontend
npm run dev
```

### 2. 访问学习系统

打开浏览器访问：
```
http://localhost:3000/exam
```

### 3. 开始学习之旅

#### 第一步：查看学习仪表板
- 了解你的当前等级和进度
- 查看可用的学习路径
- 设定学习目标

#### 第二步：选择学习模式

**模式一：系统学习（推荐新手）**
1. 点击"知识点学习"
2. 从"安装与配置"分类开始
3. 按顺序完成每个知识点
4. 完成后解锁下一个知识点

**模式二：闯关模式（推荐有基础）**
1. 点击"闯关地图"
2. 从"新手村"开始挑战
3. 完成所有必修知识点
4. 通过关卡挑战获得奖励

**模式三：模拟考试（检验学习成果）**
1. 点击"模拟考试"
2. 选择适合的难度
3. 在规定时间内完成所有题目
4. 查看成绩和答案解析

## 🎯 学习技巧

### 1. 设定明确目标
- 每天学习1-2个知识点
- 每周完成1个关卡
- 1个月内通过所有模拟考试

### 2. 实践为主
- 不要只看理论，动手实践
- 每个知识点都要完成相关挑战
- 使用实际的ES环境测试

### 3. 利用提示系统
- 遇到困难先看提示
- 理解原理比记答案重要
- 查看答案解析加深理解

### 4. 定期复习
- 完成一个分类后复习一次
- 错题要重做
- 关键知识点做笔记

### 5. 模拟考试策略
- 先做简单题，再做难题
- 注意时间分配
- 不确定的题目先标记，最后再做

## 💡 常见问题

### Q: 我是完全零基础，应该从哪里开始？
A: 从"知识点学习"的"安装与配置"分类开始，按顺序学习。

### Q: 实践题如何验证答案？
A: 系统会自动执行你的ES查询并验证结果。确保你的ES实例正在运行。

### Q: 可以跳过某些知识点吗？
A: 可以，但建议按顺序学习。某些高级知识点有前置要求。

### Q: 模拟考试可以重复做吗？
A: 可以，每次都会记录你的最佳成绩。

### Q: 学习进度会保存吗？
A: 会的，所有进度都保存在浏览器本地存储中。

### Q: 如何重置学习进度？
A: 在设置中可以找到"重置进度"选项（开发中）。
