## 🚀 立即开始使用

### 步骤1：启动项目

#### 方式A：使用启动脚本（推荐）
```bash
# Windows
start.bat

# Mac/Linux
chmod +x start.sh
./start.sh
```

#### 方式B：手动启动
```bash
# 终端1 - 启动后端
cd backend
mvn spring-boot:run

# 终端2 - 启动前端
cd frontend
npm run dev
```

### 步骤2：访问学习系统
打开浏览器访问：
```
http://localhost:3000/exam
```

### 步骤3：开始学习
1. 查看学习仪表板
2. 选择学习模式（知识点/闯关/模拟考试）
3. 完成挑战，获得经验值
4. 解锁成就和称号

## 🛠️ API接口

### 获取用户进度
```http
GET /api/exam/progress/{userId}
```

### 提交挑战答案
```http
POST /api/exam/challenge/submit
Content-Type: application/json

{
  "challengeId": "challenge-001",
  "answer": "A",
  "code": "GET /index/_search {...}",
  "timeSpent": 120
}
```

### 完成知识点
```http
POST /api/exam/topic/{topicId}/complete
```

### 完成关卡
```http
POST /api/exam/level/{levelId}/complete
```

### 重置进度
```http
POST /api/exam/progress/reset
```

## 🔧 扩展和定制

### 添加新知识点
编辑 `frontend/src/data/examTopics.ts`：
```typescript
{
  id: 'new-topic-001',
  title: '新知识点标题',
  category: 'search',
  difficulty: 'intermediate',
  description: '知识点描述',
  keyPoints: ['要点1', '要点2'],
  estimatedTime: 60
}
```

### 添加新挑战
编辑 `frontend/src/data/examChallenges.ts`，参考 `examChallengesTemplate.md`

### 自定义样式
修改组件中的样式配置，或使用Ant Design主题定制

## 🐛 常见问题

### Q: 页面打不开？
A: 确保前后端都已启动，检查端口是否被占用

### Q: 进度没有保存？
A: 检查浏览器是否禁用了LocalStorage

### Q: 代码编辑器不显示？
A: 这是正常的，编辑器使用动态加载，稍等片刻

### Q: 如何重置学习进度？
A: 调用 `/api/exam/progress/reset` 接口

### Q: 可以添加自己的题目吗？
A: 可以！参考 `examChallengesTemplate.md` 文档
